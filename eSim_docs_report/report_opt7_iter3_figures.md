**Steps:**
1. Produce `*_Comparative_Analysis_Report.csv` files inside `MonteCarlo_N60_*` folders for Toronto, Montreal, and Winnipeg (needed by 4.3.1 and 4.3.2)
2. Run full N=60 MC simulations for the 2010 and 2022 code scenarios in all three cities to generate `eplusout.sql` files (needed by 4.3.3 and 4.3.4)
3. Re-run `interim_report_gen.py` to regenerate all 4.3.x figures with complete data

**Expected result:** All four 4.3.x figures show data for all 6 scenarios across all 3 cities; no empty panels or skipped scenarios.

**Test method:** Visual inspection of regenerated PNGs; confirm each subplot has bars/lines for all 6 scenarios; confirm `interim_summary.txt` lists no MISSING entries.
