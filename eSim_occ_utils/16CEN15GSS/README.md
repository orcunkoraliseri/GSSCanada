# 16CEN15GSS Pipeline

Occupancy modeling pipeline for **Census 2016 + GSS 2015** integration.

## Structure

This folder will contain modules for:
1. **Alignment**: Harmonize Census 2016 with GSS 2015
2. **Profile Matching**: Assign GSS schedules to Census agents
3. **HH Aggregation**: Convert to 5-minute household grids
4. **BEM Conversion**: Generate hourly schedules for EnergyPlus

## Output Folder

`0_Occupancy/Outputs_16CEN15GSS/`
