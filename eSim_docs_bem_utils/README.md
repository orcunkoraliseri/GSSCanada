# Documentation

This folder contains documentation for the BEM Integration Module.

## Files

- **`implementation_plan.md`**: Technical implementation plan detailing the module structure, proposed changes, and verification strategy.
- **`walkthrough.md`**: User guide explaining how to use the BEM tool, including setup, execution, and key features.

## How to View

These files are written in **Markdown** format (`.md`). To view them with proper formatting:

### In VS Code:
1. Open the `.md` file
2. Press **`Cmd+Shift+V`** (Mac) or **`Ctrl+Shift+V`** (Windows/Linux)
   - OR right-click the file tab → **"Open Preview"**
   - OR click the preview icon (📖) in the top-right corner

### On GitHub:
Markdown files are automatically rendered when viewed in the repository.

### In a Web Browser:
You can use online Markdown viewers like [dillinger.io](https://dillinger.io) or install a browser extension.
